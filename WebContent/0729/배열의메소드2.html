1. slice/splice

- slice(start, end) : start ~ end -1까지의 요소들을 추출
- splice(idx, n, a, b, c) ; index위치에서 n개를 삭제하고 a,b,c요쇼ㅗ를 추가





2. concat/join/reverse

- a.concat(b, c,...) : 여러개의 배열 요소를 한개의 배열로 합친다
- a.join() : 한개의 배열요소를 하나의 문자열로 변환한다.
- a.reverse() : a배열의 순서를 뒤집기








3. sort

arr = [234, 21, 150, 11, 55, 501]
- a.sort() : 11 150 21 234 501 55
* 앞자리부터 순서대로 정렬함. 1 -> 2 -> 3,... - ASC : 11 21 55 150 234 501 - DESC : 501 234 150 55 21 11
